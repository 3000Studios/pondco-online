# pondco-online# PondCo Online

Professional project configuration for Airport Traffic Control Towers (ATCT) contracts.

## Project Scope
This project manages the configuration and online presence for Airport Traffic Control Towers contracts at the following locations:
- **French Valley Airport (F70)**
- **Jacqueline Cochran Regional Airport (TRM)**

## Configuration Details
- Single-page application framework.
- Professional services for ATCT operations.
- Real-time monitoring and reporting.
